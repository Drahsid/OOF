# OOF!

Dalamud plugin that plays the oof sound when you take 
- fall damage
- die
- party/alliance members die
- run /oof command

#### play oof sound:
```
/oof
```
#### change settings:
```
/oofsettings
```


## install
now in the dalamud repository! 

open dalamud plugins options > turn on testing builds
